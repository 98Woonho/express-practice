# Express에 mysql 연동

## 연동 방법
1. 터미널에 `npm install mysql --save` 입력해서 설치

2. <b>db.js</b> 파일 생성 후 연동 할 db 정보 작성. (폴더 내 <b>db.js</b> 참조)
